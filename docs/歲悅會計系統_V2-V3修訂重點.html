歲悅會計系統 V2 → V3 修訂重點

本文件整理 V3 相較 V2 的主要修訂方向，供教育訓練、測試回饋與正式上線前檢核使用。

一、流程穩定性

  • 強化暫存、送出簽核、抽單取消、退回上一關與加簽流程。
  • 申請送出後會先保留本機備援，降低 Supabase 暫時寫入失敗時畫面消失的風險。
  • 正式登入與測試登入都會先載入本機備援資料，再進行遠端同步。

二、附件與憑據

  • 強化主管檢核附件下載，若 Supabase signed URL 失敗，會改用備援下載連結。
  • 附件依申請單號、資料環境、法人、部門、日期與檔案類型歸檔。
  • 支援憑據照片彙整、即時 Excel 明細、存摺封面等不同附件類型分開保存。

三、報表與稅務

  • 三表改以正式分類帳為主要資料來源，並提供來源追溯。
  • 發票收入拆分未稅收入與銷項稅額，支出入帳拆分費用與進項稅額。
  • 新增進項營業稅、銷項營業稅與可折抵稅額檢視，避免同一單號重複計算。

四、會計合規

  • 新增月結鎖帳、傳票沖銷 / 調整、三表試算平衡與 audit log 匯出。
  • 傳票使用正式流水號，過帳後不可直接修改，只能透過沖銷或調整處理。
  • 會計科目與系統設定異動加入版本紀錄與核准概念。

五、收入與股東往來

  • 收入申請整合股東往來、開立發票、申請繳費單與收款情形。
  • 股東出資 / 增資、股東借款與償還股東借款分別對應正式會計科目。
  • 股東往來不列收入或費用，依性質進入資產負債表與現金流量表。
